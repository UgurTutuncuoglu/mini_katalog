# mini_katalog

Kısa Açıklama
--------------
`mini_katalog`, API üzerinden ürünleri çekip listeleyen, ürün detayını gösteren, sepete ekleme ve ödeme adımlarını destekleyen basit bir Flutter katalog uygulamasıdır.

Flutter Sürümü
---------------
- Flutter Version: 3.41.9
- Dart Version: 3.11.5
- DevTools Version: 2.54.2

Çalıştırma Adımları
-------------------
1. Flutter SDK'nın kurulu olduğundan emin olun:
   ```bash
   flutter --version
   ```

2. Projeyi bilgisayarınıza klonlayın veya indirin.

3. Proje klasörüne gidin:
   ```bash
   cd mini_katalog
   ```

4. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```

5. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```

Ekran Görüntüleri
-----------------
- Ana ekran
- Ürün detay ekranı
- Sepet ekranı
- Ödeme ekranı
